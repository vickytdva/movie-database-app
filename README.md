# Movie Database Web Application

This is a full-stack web application developed for CIC6003 Advanced Programming.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## Features
- User login
- Add movies
- Rate movies
- Add comments
- View statistics by user demographics

## How to Run
1. Import `database.sql` into MySQL Workbench.
2. Open the project folder in terminal.
3. Run:
   npm install
4. Start the server:
   node server.js
5. Open:
   http://localhost:3000/login.html
